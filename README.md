# OjekHub MVP

Applications for connecting farmers/warehouses with local labor in Rejang Lebong.

## Structure

- `backend/`: NestJS API (Vercel Serverless)
- `frontend/`: Flutter Android App
- `docs/`: Planning documentation

## Tech Stack

- **Frontend**: Flutter (Android)
- **Backend**: NestJS
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth (Google)

## Setup

See individual `README.md` files in `backend` and `frontend` directories for setup instructions.
