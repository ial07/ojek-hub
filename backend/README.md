# OjekHub Backend

NestJS API for OjekHub, deployed on Vercel.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run locally:
   ```bash
   npm run start:dev
   ```

## Environment Variables

Create a `.env` file:

```
SUPABASE_URL=your_project_url
SUPABASE_KEY=your_anon_key
JWT_SECRET=your_jwt_secret
```

## Deployment

Deploy to Vercel:

```bash
vercel
```
